## AGENDA RETROSPECTIVA SEMANAL (60 MIN)

### Estructura Retrospectiva

| Tiempo | Tema | Facilitador |
|--------|------|-------------|
| 00-05 min | Numeros de la semana | Socio 3 |
| 05-15 min | Que salio bien? (Celebrar) | Todos |
| 15-30 min | Que no salio como esperabamos? | Todos |
| 30-45 min | Que aprendimos? Que ajustamos? | Socio 1 |
| 45-55 min | Prioridades semana siguiente | Todos |
| 55-60 min | Compromiso personal de cada uno | Cada persona |

### Preguntas Guia Retrospectiva

**Wins de la semana:**
- Cuantas demos hicimos?
- Cuantos cierres?
- Algun cliente super contento?
- Que nos funciono mejor para agendar demos?

**Problemas/Retos:**
- Que nos costo mas esta semana?
- Algun deal que perdimos? Por que?
- Donde perdimos tiempo?
- Que nos bloqueo?

**Learnings:**
- Que hariamos diferente?
- Que proceso hay que ajustar?
- Algun insight de clientes/prospectos?

**Siguiente semana:**
- Top 3 prioridades?
- Que demos ya tenemos confirmadas?
- Hay algo urgente que atender?

## BLOQUES DE ACTIVIDADES ESTANDAR

### BLOQUE PROSPECTING (2 HORAS)

**Objetivo**: 10-15 llamadas por persona

**Estructura del bloque:**
```
00-10 min: Revisar lista de prospectos, priorizar
10-20 min: Llamada 1 + notas
20-30 min: Llamada 2 + notas
30-40 min: Llamada 3 + notas
40-50 min: Llamada 4 + notas
50-60 min: Llamada 5 + notas
60-70 min: Llamada 6 + notas
70-80 min: Llamada 7 + notas
80-90 min: Llamada 8 + notas
90-100 min: Enviar emails a los que no contestaron
100-120 min: Actualizar CRM, notas
```

**Metricas del bloque:**
- Llamadas realizadas: ____
- Conexiones (hablo con decision maker): ____
- Demos agendadas: ____
- Seguimientos prometidos: ____

**Tips:**
- Sonreir antes de cada llamada (se nota en la voz)
- Tener script visible pero no leerlo
- Notas en CRM INMEDIATAMENTE despues de cada llamada
- Si no contesta 3 veces, pasar a email

---

### BLOQUE DEMOS (1 HORA POR DEMO)

**Estructura de Demo (60 minutos):**
```
00-05 min: Saludo + preguntas contexto
05-10 min: Entender dolor especifico del prospecto
10-35 min: Demo en vivo del producto
35-45 min: Casos de uso especificos para SU negocio
45-52 min: Propuesta comercial (piloto 50% OFF)
52-60 min: Proximos pasos + cierre
```

**Preparacion Pre-Demo (15 min antes):**
- [ ] Revisar notas del prospecto en CRM
- [ ] Personalizar ejemplos para su industria
- [ ] Verificar que demo environment funciona
- [ ] Tener propuesta lista para enviar post-demo

**Post-Demo (15 min despues):**
- [ ] Actualizar CRM con notas de la demo
- [ ] Enviar email con propuesta (dentro de 2 horas)
- [ ] Agendar seguimiento en calendario
- [ ] Debrief rapido con equipo si relevante

**Metricas:**
- Demo completada: Si/No
- Interes mostrado (1-5): ____
- Objeciones principales: ____
- Siguiente paso acordado: ____
- Probabilidad de cierre (%): ____

---

### BLOQUE SEGUIMIENTO (1 HORA)

**Objetivo**: Contactar todos los prospectos tibios/calientes

**Orden de prioridad:**
1. Prospectos con propuesta enviada (calientes)
2. Post-demos sin respuesta (tibios altos)
3. Demos agendadas para confirmar
4. Prospectos que pidieron mas info
5. Reactivaciones (prospectos frios)

**Canales a usar:**
- WhatsApp: Para seguimientos rapidos, confirmaciones
- Llamada: Para cierre, objeciones, negociacion
- Email: Para enviar info, propuestas, formalizacion

**Template seguimiento WhatsApp:**
```
Hola [Nombre], como estas?
Te escribo para ver si tuviste oportunidad de revisar
la propuesta que te envie el [dia].
Quedo atento a cualquier duda. Que tal si agendamos
una llamada rapida de 15 min para conversarlo?
```

**Metricas del bloque:**
- Seguimientos realizados: ____
- Respuestas obtenidas: ____
- Cierres logrados: ____
- Demos re-agendadas: ____

---

### BLOQUE ONBOARDING CLIENTE (1-2 HORAS)

**Objetivo**: Dejar cliente configurado y funcionando

**Participantes**: Socio 2 (tecnico) + Socio 3 (CS)

**Estructura Onboarding:**
```
Sesion 1 (60-90 min):
00-10 min: Bienvenida, explicar proceso
10-30 min: Recopilar info: productos, precios, horarios
30-60 min: Configuracion inicial del chatbot
60-90 min: Pruebas en vivo + ajustes

Sesion 2 (30-60 min, dia siguiente):
00-20 min: Revisar primeras interacciones
20-40 min: Ajustes finos segun feedback
40-60 min: Training al cliente: como ver reportes
```

**Checklist Onboarding:**
- [ ] WhatsApp Business conectado
- [ ] Catalogo de productos cargado
- [ ] Precios configurados
- [ ] Horarios de atencion definidos
- [ ] Mensaje de bienvenida personalizado
- [ ] Prueba de conversacion exitosa
- [ ] Cliente sabe ver reportes
- [ ] Canal de soporte claro (WhatsApp a Socio 3)

**Metricas:**
- Tiempo total onboarding: ____
- Cliente satisfecho (1-5): ____
- Configuracion completa: Si/No
- Primeras conversaciones exitosas: ____
