## CALCULADORA RÁPIDA PARA PROSPECTO

### Usa esta fórmula en la llamada:

**Paso 1: Cuantifica el tiempo**
```
"¿Cuántas consultas recibes por WhatsApp al día?"
Respuesta: [X] consultas

"¿Cuánto tiempo toma responder cada una?"
Respuesta: [Y] minutos

Cálculo: X × Y × 26 días = [Z] minutos/mes = [Z/60] horas/mes
```

**Paso 2: Calcula costo actual**
```
"¿Cuánto gana aproximadamente la persona que responde?"
Respuesta: $[A]/mes

Cálculo hora: $[A]/mes ÷ 176 horas = $[B]/hora
Costo mensual: [Z/60] horas × $[B]/hora = $[C]/mes
```

**Paso 3: Calcula ingresos perdidos**
```
"¿Cuántas consultas llegarán fuera de horario que no respondes?"
Respuesta: [D]/mes

"¿Cuál es el valor promedio de una venta?"
Respuesta: $[E]

"De esas consultas que no respondes, ¿qué % crees que compraría?"
Respuesta: [F]%

Ingresos perdidos: [D] × $[E] × [F]% = $[G]/mes
```

**Paso 4: Muestra el ROI**
```
Beneficio total con Tintero:
Ahorro tiempo: $[C]
Ingresos recuperados: $[G] (asumiendo 50% de recuperación)
TOTAL: $[C + G/2] por mes

Inversión mes 1: $53.550 (Fase 1) o $77.350 (Fase 2)

ROI: [TOTAL / Inversión] × 100 = [X]%
Payback: [Inversión / (TOTAL/30)] días
```
